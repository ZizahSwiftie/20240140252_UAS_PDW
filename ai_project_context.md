# Public Complaint Management System

## Stack
- Pure procedural PHP
- Bootstrap 5
- MySQL
- XAMPP localhost
- Leaflet.js
- OpenStreetMap
- Nominatim API

## Architecture Rules
- No frameworks
- No Laravel
- No React/Vue
- No OOP PHP
- Use mysqli only
- Use prepared statements
- Use snake_case naming
- Keep beginner-friendly architecture
- Maintain existing folder structure

## Existing Features
- Authentication system
- Role-based access
- Complaint submission
- Secure image upload
- User dashboard
- My Reports
- Admin dashboard
- Admin report management
- Status tracking
- Admin responses
- Category filtering
- Geolocation picker
- GPS auto-location
- Nominatim keyword search
- Profile management
- Responsive glassmorphism-inspired UI

## Security Rules
- Use htmlspecialchars()
- Use prepared statements
- Validate uploads
- Validate ownership
- Protect admin routes

## UI Style
- Modern professional government portal style
- Slight transparency
- Subtle animations
- Bootstrap-based
- Responsive layout

## Restrictions
- Do not redesign architecture
- Do not switch frameworks
- Do not convert to OOP
- Do not replace Bootstrap
- Do not add unnecessary libraries
